Assignment:

The assignment consists of 6 exercise.Exercise 1(exercise-1-typescript-warmup) is independent rest other are connected (trader-desktop).The server part is being handled by 'trader-server'.

Steps to run the assignment:

1> Run 'yarn install' in all the folders i.e in exercise-1-typescript-warmup, trader-desktop, trader-server.
2> Run 'yarn build' in all the folders.
3> Run 'yarn start' to start the apps.


* In case yarn build fails throwing linebreak-style error use command 'npm run lint -- --fix' to fix issue and then run 'yarn build'.

* Make sure that trader-desktop and trader-server are both running while testing the application.
